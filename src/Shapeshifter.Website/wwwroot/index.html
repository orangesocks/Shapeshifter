Shapeshifter

a clipboard manager for the 21st century

Why?

  • It doesn't change your existing habits CTRL + C to copy, CTRL + V to paste, hold down CTRL + V and use the arrow keys to swap between items.
  • It looks sexy The UI is tailored to adapt to your Windows 10 color scheme, and is designed to be minimal and efficient.
  • It is maintained by you Powered by open source, anyone can help improve it. As a result, we keep it updated constantly.
